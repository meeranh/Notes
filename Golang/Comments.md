## Line Comments
+ These are comments **only for a single line**.

```go
package main

import "fmt"

func main(){
    fmt.Println("Hello World") // This is a line comment
}
```

## Block Comments
+ These are comments **for multiple lines**.

```go
package main

import "fmt"

func main(){
    /*
    This is a block comment
    It can span multiple lines
    */
    fmt.Println("Hello World")
}
```
